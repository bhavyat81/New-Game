# 🔨 Hardware Havildar

Indian Hardware Store Tycoon — built in Godot 4

## Run the game

```bash
# Install Godot 4 (Mac)
brew install --cask godot

# Clone and open
git clone https://github.com/bhavyat81/New-Game.git
cd New-Game
godot project.godot
```

Press **F5** to play.

## Game Loop
- Customers walk in → go to shelf → pick item → walk to billing
- You stand at billing counter → coins collected
- Shelf empty → go to godown → walk over items → restock shelf

## Controls
- Virtual joystick (bottom-left) to move player
- Walk into zones to interact automatically
